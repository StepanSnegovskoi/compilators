# Pascal Compiler and AST Interpreter

Проект представляет собой компилятор и интерпретатор подмножества языка Pascal, реализованный на языке Python. Программа осуществляет полный цикл обработки кода: от лексического анализа до выполнения в виртуальной среде.

## Функциональные возможности

* **Типы данных:** integer, boolean, char.
* **Структуры:** одномерные массивы.
* **Операции:** полный набор арифметических и логических операций, включая div и mod.
* **Управляющие конструкции:** if-then-else, while, do-while, for.
* **Подпрограммы:** поддержка процедур с передачей параметров по значению.
* **Встроенные функции:** Write/WriteLn, Read/ReadLn, Inc, Dec, Abs.

## Этапы разработки

### 1. Синтаксический анализ
* Разработана формальная грамматика (EBNF).
* Реализован парсер на базе библиотеки Lark, обеспечивающий построение абстрактного синтаксического дерева (AST).
* Реализована диагностика синтаксических ошибок.

### 2. Семантический анализ
* Реализована таблица символов для управления областями видимости.
* Выполняется проверка типов выражений и соответствия аргументов подпрограмм.
* Производится модификация AST (например, приведение типов для целочисленного деления).

### 3. Оптимизация и исполнение
* Constant Folding: свертка константных выражений на этапе компиляции.
* Dead Code Elimination: удаление недостижимых блоков кода и циклов с ложным условием.
* Интерпретатор: рекурсивный обход оптимизированного AST с эмуляцией стека вызовов.

## Структура проекта

* grammar.py — описание грамматики и парсер.
* semantic.py — семантический анализатор и контроль типов.
* optimizer.py — модуль оптимизации AST.
* interpreter.py — интерпретатор команд.
* main.py — точка входа и автоматизированный запуск тестов.
* examples/ — папка с исходными кодами программ на Pascal для тестирования.

## Установка и запуск

Для работы требуется установленный Python 3.7+.

1. Установка зависимостей:
   pip install lark

2. Запуск компиляции всех примеров из папки examples/:
   python main.py
